# Audio Transcriber

A Python-based audio transcription tool that uses OpenAI's Whisper model (via faster-whisper) to automatically transcribe audio files to text with intelligent paragraph formatting.

## Supported Audio Formats

- MP3
- WAV
- M4A
- FLAC

## Requirements

- Python 3.8+
- ffmpeg (for audio processing)

## Installation

1. **Clone or download this repository**

2. **Create a virtual environment** (recommended):
   ```bash
   python -m venv .venv


## Run Transcriber

1. **Put audio file under `audio` folder

2. **Run `python main.py`
